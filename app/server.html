E-Mail-Server

Der Server kann die E-Mails im Klartext lesen, sofern sie nicht mit PGP verschlüsselt wurden.

SSL schützt den Übertragungsweg zwischen den einzelnen Rechnern und Servern. Für die Zustellung muss der Server jedoch wissen, wer wem die E-Mail geschickt hat.

Geheimdienste haben oft direkten Zugriff auf die E-Mail-Server und können so die Mails mitlesen.

Dies geschieht entweder über ausgenutzte Sicherheitslücken oder über eine so genannte SINA-Box. (Siehe §110 TKG)
